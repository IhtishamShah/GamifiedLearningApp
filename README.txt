Pyhton 2.7 and pip required:
	run "pip install -r requirements.txt"
For all the dependencies related to front-end(react js):
	"npm install" 
This will install all the dependencies on the packages.json file.


In order to run the code:
	- go to the project folder and type:
		"python manage.py runserver"
	- The open web browser and go to:
		"localhost: 8000/main"
